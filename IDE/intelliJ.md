#인텔리J

### 이클립스 프로젝트를 인텔리J로 가져오는 방법?
> 이직한 회사에서 인텔리J ultimate 버전을 구매해줘서 나도 드디어 인텔리J를 사용해 본다. 기존에 작업하던
> 이클립스 프로젝트를 그대로 git clone 받아서 사용하려고 하니 error가 떡하고 나타났다. 
> 좋은 블로그 글을 찾았기 때문에 굳이 정리하지 않고 공유로 대체한다.
